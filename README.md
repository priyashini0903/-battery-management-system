# Smart IoT-Based Battery Management System

An IoT-enabled Battery Management System to monitor battery health in real time.

## Features
- Real-time monitoring of voltage, current, temperature, SOC and SOH
- Automatic relay cut-off to prevent unsafe conditions
- Flask-based web dashboard with live parameters and alerts
- AI-based anomaly detection using Isolation Forest
- SOC and SOH prediction using Random Forest

## Technologies Used
- ESP8266, INA219, DHT11 sensors
- Python, Flask
- HTML, CSS, JavaScript
- Machine Learning (Random Forest, Isolation Forest)
